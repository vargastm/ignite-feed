### 🎓 Instructions

**Clone** the repository

```bash
git https://github.com/vargastm/ignite-feed.git
```

<br>

To run **Ignite Feed** - Open the **ignite-feed** folder with the terminal and **run** the commands:

```bash
npm install
npm run dev
```

> Server will be running on localhost:3000
